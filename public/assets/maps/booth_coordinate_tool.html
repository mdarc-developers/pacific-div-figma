Booth Coordinate Capture Tool

Instructions:

  1. Load the image using the file selector below
  2. Enter booth name (e.g., "1", "2", "NP-1") in the input field
  3. Click 2 corners of the booth: top-left, then bottom-right
  4. The tool will automatically calculate all 4 corners and output coordinates in the format: [[y1, x1], [y2, x2], [y3, x3], [y4, x4]]
  5. Y-coordinates are measured from the bottom of the image
  6. Click "Clear Last Point" to undo, or "Reset Booth" to start over
  7. Copy the generated code from the output section
