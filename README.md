
  # Attendee Conference App

  This is a code bundle for Attendee Conference App. The original project is available at https://www.figma.com/design/fAJt1K7Tm5xlgtypyfRTq5/Attendee-Conference-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to prepare for deployment to firebase
  
  Run `npm run deploy` to copy to firebase  and https://pacific-div.web.app
